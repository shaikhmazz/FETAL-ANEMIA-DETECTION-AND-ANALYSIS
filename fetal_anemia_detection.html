How It Works

Detection Pipeline

End-to-end flow from raw ultrasound input to clinical classification output, derived from the GeoProteoNet + Random Forest two-stage model.

01

Image Acquisition & Preprocessing

Doppler ultrasound image is loaded and resized to 128×128 px. Grayscale normalization is applied (pixel values scaled 0–1 using OpenCV). Accepts DICOM, PNG, JPG, BMP, TIFF formats.

cv2.imread() cv2.resize(128,128) / 255.0
02

GeoProteoNet Segmentation (CNN)

The custom encoder-decoder CNN (GeoProteoNet) segments fetal abdominal structures from the ultrasound. The architecture uses Conv2D → MaxPooling → UpSampling layers with sigmoid output to produce a binary segmentation mask.

Conv2D(64/128/256) MaxPooling2D UpSampling2D sigmoid output
03

Hemodynamic Feature Extraction

Five vascular biomarkers are computed from the segmentation mask. These mimic Doppler waveform analysis used in clinical MCA assessment for fetal anemia screening.

PSV = max(mask) EDV = min(mask) RI = (PSV−EDV)/PSV PI = (PSV−EDV)/mean S/D = PSV/EDV
04

Random Forest Classification

The extracted 5-feature vector is fed into a trained Random Forest Classifier (100 estimators). Features are standardized using StandardScaler. The classifier outputs a binary label: Anemia Detected / Normal.

RandomForestClassifier(n=100) StandardScaler stratified split
05

Report Generation

Results include risk score, individual hemodynamic metrics, feature correlation context, waveform plots, and clinical recommendation. Exportable as tabulated report (tabulate library).

classification_report confusion_matrix tabulate

STAGE 1 — GeoProteoNet CNN

Encoder-Decoder Segmentation

  • Input: (128, 128, 1) grayscale ultrasound
  • Encoder: Conv2D(64) → MP → Conv2D(128) → MP → Conv2D(256)
  • Decoder: UpSampling2D → Conv2D(128) → UpSampling2D → Conv2D(64)
  • Output: Conv2D(1, sigmoid) binary mask
  • Loss: Binary Crossentropy | Optimizer: Adam (lr=0.001)
  • Training: 15 epochs, batch size 16
  • Transfer learning variant: 224×224, ImageDataGenerator augmentation
STAGE 2 — Random Forest Classifier

Hemodynamic Risk Classification

  • Input: 5-feature vector [PSV, EDV, RI, PI, S/D]
  • n_estimators = 100, random_state = 42
  • Train/test split: 80/20 stratified
  • Preprocessing: StandardScaler normalization
  • Output: Binary (0 = Normal, 1 = Anemia)
  • Evaluation: precision, recall, F1, confusion matrix

Hemodynamic Biomarker Reference

FeatureFormulaClinical MeaningAnemia Indicator
PSV max(mask pixel intensity) Peak Systolic Velocity — max blood flow speed PSV > 1.5 MoM = High Risk
EDV min(mask pixel intensity) End Diastolic Velocity — min blood flow speed Low EDV = vessel abnormality
RI (PSV − EDV) / PSV Resistive Index — downstream vascular resistance RI → 1.0 in fetal anemia
PI (PSV − EDV) / mean(PSV,EDV) Pulsatility Index — waveform variability PI → 2.0 in severe cases
S/D PSV / EDV Systolic-to-Diastolic ratio Very high S/D = critical risk

# app.py from flask import Flask, request, jsonify from flask_cors import CORS import cv2, numpy as np from joblib import load from tensorflow.keras.models import load_model app = Flask(__name__) CORS(app) cnn = load_model('geoproteonet.h5') clf = load('trained_model.joblib') scaler= load('scaler.joblib') @app.route('/predict', methods=['POST']) def predict(): file = request.files['image'] nparr = np.frombuffer(file.read(), np.uint8) img = cv2.imdecode(nparr, cv2.IMREAD_GRAYSCALE) img = cv2.resize(img, (128,128)) / 255.0 img = img.reshape(1,128,128,1) mask = cnn.predict(img) mask = (mask > 0.5).astype(np.uint8) feats = extract_features(mask.squeeze()) arr = scaler.transform([list(feats.values())]) label = clf.predict(arr)[0] return jsonify({'result': int(label), 'features': feats}) if __name__ == '__main__': app.run(debug=True, port=5000)

Project Information

About This System

This system is the major project capstone for B.E. VIII Semester Computer Science Engineering at Muffakham Jah College of Engineering and Technology (MJCET), Hyderabad. It presents a non-invasive AI-assisted approach to fetal anemia screening using Doppler ultrasound imaging.

The two-stage pipeline combines a custom GeoProteoNet encoder-decoder CNN for fetal abdominal structure segmentation with a Random Forest classifier operating on hemodynamic features (PSV, EDV, RI, PI, S/D ratio) extracted from the predicted segmentation masks.

The dataset used is the Fetal Abdominal Structures Segmentation Dataset Using Ultrasonic Images. Model training was performed on Google Colab with TensorFlow 2.x. Image augmentation (rotation, shear, zoom, flips) was applied in the transfer learning variant to improve generalization.

⚠️ Medical Disclaimer: This system is a research prototype developed for academic purposes. It is not a certified medical device and must not be used as a sole diagnostic tool. All results should be verified by a qualified medical professional (perinatologist, sonographer). The model operates in demo simulation mode in this web interface — live inference requires the Python backend with trained model files.
